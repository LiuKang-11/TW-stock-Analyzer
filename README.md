# TW-Stock Analyzer
### A feature-driven filtering tool to help investors identify promising Taiwan stocks

## Overview
TW-Stock Analyzer is a lightweight analytical tool designed for retail investors who want to filter Taiwan stock candidates based on customized technical, fundamental, or chip-based (籌碼面) features.  
The project simplifies the discovery process by transforming open-source financial data into structured, investor-friendly signals.
